# mattermost-bsky-plugin for Linux/Arm64
a better unfurl for bluesky posts in Mattermost on Linx/Arm64 machines (like the Raspberry Pi I run mine on)

This was vibe coded, so use at your own risk. I am technically inclined, but not an expert software engineer by any stretch.

![Screenshot](https://github.com/asimone/mattermost-bsky-plugin/blob/main/assets/screenshot.png?raw=true)

## Installation

1. Download [bluesky-unfurl.tar.gz](../../raw/main/bluesky-unfurl.tar.gz)
2. In Mattermost go to **System Console → Plugin Management → Upload Plugin** and upload the file
3. Enable the plugin — no configuration required (engagement counts can be toggled in the plugin settings)
