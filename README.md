## Installation

1. Download [bluesky-unfurl.tar.gz](../../raw/main/bluesky-unfurl.tar.gz)
2. In Mattermost go to **System Console → Plugin Management → Upload Plugin** and upload the file
3. Enable the plugin — no configuration required (engagement counts can be toggled in the plugin settings)

# How to use

1.) Download the [bluesky-unfurl.tar.gz](https://github.com/6984e416-d1fa-4835-8eec-e97f832ac28c). 

2.) In the main menu in the upper left go to System Console > Plugin Management and upload the package.

3.) Go to the plugin settings, enable the package and toggle "Show Engagement Counts" on/off (true/false) depending on preferences.
